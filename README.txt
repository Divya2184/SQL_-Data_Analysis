-- Power BI Dashboard - Sales & Financial Analysis :-
   ================================================

  -- Overview :
     ---------

       - Interactive Power BI dashboard showing Sales, Profit, Units Sold, and Gross Sales by Country, Product, and Date.
         Designed to help stakeholders explore trends and insights quickly.

  -- Visuals :
     --------

     - Cards → Total KPIs  
     - Line Chart → Sales by Month  
     - Column Chart → Product-wise Sales  
     - Map → Country-wise Sales  
     - Treemap → Profit by Product  

  -- Interactivity :
     -------------

     - Slicers: Country, Product Category, Date  
     - All visuals update dynamically with slicers  

  -- Tools & Dataset :
     ---------------

      - Power BI Desktop
 
      - Sales_Financial dataset
